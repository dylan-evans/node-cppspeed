# Node.js modules speed test: C++ vs JavaScript

Download, build and test instructions:

    $ git clone git://github.com/lupomontero/node-cppspeed.git
    ...

    $ cd node-cppspeed
    $ node-waf configure build
    ...

    $ node test/cppspeed.js

    foo run in 167ms
    cppfoo run in 18ms
    c++ was 9.3 times faster
